Jogo da Velha

O Projeto

Modalidade : GAME

O Projeto visa desenvolver um jogo na plataforma Allegro de criação de games com base na linguagem C. De uma forma descontraida e divertida, brincando com os personagens de duas velhas que disputam uma partida de jogo da velha.

Utilizamos o conceito da própria historia do jogo da velha que surgio na Inglaterra no século 19, quando mulheres se reuniam nos finais de tarde para conversar e bordar.E as mais idosas, por não conseguirem mais bordar em razão de suas vistas fracas, se entretiam com o jogo que passou a ser chamado “noughts and crosses” (“nós e cruzes”).

No entando adaptamos a uma perspectiva onde os jovens lembram da sua vó, sendo aquela vó simpática que sempre faz comidas deliciosas, tais como bolos e muitas vezes com sua simpática begala que a ajuda a caminhar.

Dentro do desenvolvimento do projeto utilizamos as ferramentas :

  • GIT
  • Code Blocks
  • DEV C++
  • Photoshop CS6
  • TFF2PXC (conversor de fontes para o allegro)

Objetivo

O obejtivo é projetar um jogo da velha na plataforma C, usando a biblioteca Allegro.

Desta forma desenvolvendo os métodos e conceitos de criação de jogos tanto para computadores, mas também visando o uso desde para consoles de video games.
